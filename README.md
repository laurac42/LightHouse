LightHouse User Guide

Sign Up
1.	Access LightHouse at the following URL: light-house-l9rhkdmyb-lauras-projects-fce00650.vercel.app. You will be taken to the website home page.
2.	Click Sign In at the top right hand corner to create an account, signing up with a username and password. 
a.	Click the link in the verification email you receive to verify your account and sign in 
 
3.	Complete the onboarding steps
a.	Enter your name and set your buyer goals. (Note: to receive personalised property recommendations, you must select buy as a buyer goal)
b.	Enter your buyer preferences 
c.	Enter any locations important to you, such as your workplace, or a friend or family’s house
Search for Properties
1.	Search for properties in a specific location from the site home page. (Note: the platform only contains properties in Tayside and Fife) 
2.	Select and apply your chosen filters
 
3.	View the property search results
a.	Note that if any properties matching your preferences were found, these will be marked as recommended
 
4.	Adjust your filters, including applying more advanced filters, using the More Filters button at the right hand side of the filter bar
5.	Click on a property to view the full property details 
 
6.	Use the estate agent contact details to contact the estate agent if desired

Additional Features
Favourites:
1.	Save a property as a favourite using the heart icon on the property preview or property details page
2.	View all favourites from the ‘Profile/Favourites’  button at the right hand side of the navbar
 
Tags:
1.	Scroll down the property details page to view all tags users have added to a property
 
2.	Using information from the property photos and description, upvote an existing tag you agree with, or add your own tag
3.	Report any tags as irrelevant or inappropriate using the ⋮  icon on the tag you wish to report
View and edit your profile:
1.	Click the ‘Profile/My Account’ button at the right-hand side of the navbar to view your profile
2.	Edit your personal details from the main profile page
a.	You can delete your account using the ‘Delete Profile’ button, which will sign you out and delete all of your personal information
3.	Edit your user goals, preferences and personal locations using the side navigation options
